# スクールチャイム管理システム

ブラウザ上で動作する、高機能なスクールチャイム管理システムです。ReactとTailwind CSSを使用して構築されており、ビルドプロセスなしで単一のHTMLファイルとして直接実行できます。

## ✨ 主な機能

- **多彩なスケジュール設定**:
  - **繰り返し**:「毎週月曜日と水曜日の8:30」のように、曜日を指定して毎週繰り返す設定が可能です。
  - **日付指定**:「2025年10月1日の13:00」のように、特定の年月日を指定して一度だけ鳴らす設定が可能です。
- **柔軟な音源管理**:
  - **デフォルト音源**: 標準でスクールチャイムの音源が用意されています。
  - **URL指定**: Web上の音声ファイルURLを直接登録できます。この設定はブラウザを閉じても保存されます。
  - **クラウド連携**: Google Drive, Dropbox, OneDriveの共有リンクを貼り付けるだけで、再生可能な形式に自動変換して登録します。
  - **ファイルアップロード**: PC上の音声ファイル（MP3など）を一時的にアップロードして使用することも可能です。
- **個別音量設定**:
  - スケジュールごとに音量を0%〜150%の範囲で細かく調整できます。
  - ワンクリックで100%に戻すリセットボタンも備えています。
- **直感的なUI**:
  - 現在時刻を示すアナログ・デジタル時計。
  - 次に鳴るチャイムの予定と名前を表示するステータスパネル。
  - ダークモードに対応し、OSの設定に連動して自動で切り替わります（手動切り替えも可能）。
  - 有効・無効が一目でわかるトグルスイッチ。
- **データの永続化と共有**:
  - 作成したスケジュールとURL指定した音源は、ブラウザの`localStorage`に自動で保存され、ページをリロードしても消えません。
  - **インポート/エクスポート**: 登録したスケジュール一覧をJSONファイルとして書き出し、他のPCやブラウザで読み込むことができます。

## 🛠️ 技術スタック

- **React**: UIの構築
- **Babel (Standalone)**: JSX記法をブラウザで直接解釈
- **Tailwind CSS**: UIのスタイリング
- **Lucide Icons**: アイコン表示

## 🚀 使い方

このアプリケーションを使用するのに、特別なインストールや環境構築は必要ありません。

1.  `index.html` ファイルをGoogle ChromeやFirefoxなどのモダンなウェブブラウザで開きます。
2.  これだけで、すぐにチャイムの設定を開始できます。

### 設定のステップ

1.  **ステップ1: チャイム音の設定**
    - **URLから追加（推奨）**:
      - 音声ファイルのURL、またはGoogle Driveなどの共有リンクと、任意の「表示名」を入力して「URLから音源を追加」ボタンをクリックします。この方法で追加した音源は永続的に保存されます。
    - **ファイルから追加（一時的）**:
      - 「ファイルから追加」ボタンをクリックし、PC上の音声ファイルを選択します。ここで追加した音源は、ページをリロードするとリセットされます。

2.  **ステップ2: スケジュール追加**
    - 「名前」「時間」「チャイム音」などをフォームに入力します。
    - 「タイプ」で「毎週」または「日付指定」を選択し、それに合わせて曜日や日付を設定します。
    - 必要に応じて音量を調整します。
    - 「スケジュールに追加」ボタンをクリックします。

3.  **ステップ3: 登録済みスケジュールの管理**
    - 登録済みのスケジュールが一覧で表示されます。
    - 左側のトグルスイッチで、各スケジュールの有効/無効を切り替えられます。
    - 右側のボタンで、テスト再生、編集、削除が可能です。

## ⚠️ 注意事項

- **音声ファイルの自動再生**:
  - ブラウザのセキュリティポリシーにより、ユーザーが一度画面をクリックするまで音声が自動再生されない場合があります。チャイムが鳴らない場合は、一度画面のどこかをクリックしてください。
- **ファイルアップロード**:
  - 「ファイルから追加」でアップロードした音源は、ブラウザの仕様上、ページをリロードすると消えてしまいます。設定を保存したい場合は、音源をWeb上にアップロードし、「URLから追加」機能を使用してください。
